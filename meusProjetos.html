Padaria do João

O projeto "Padaria do João" envolveu a modelagem e criação de um banco de dados relacional (MER) utilizando SQL para gerenciar os processos internos de uma padaria. O objetivo foi desenvolver uma estrutura eficiente para armazenar informações sobre produtos, clientes, vendas e estoque. Durante a implementação, criei tabelas inter-relacionadas, assegurando integridade referencial e otimizando consultas para garantir rapidez no processamento de dados. Além disso, utilizei conceitos como normalização de dados e criação de índices para melhorar o desempenho das operações, tornando o sistema robusto e escalável para atender às necessidades do negócio.
